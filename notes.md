# Repository Branching

--dev: The primary integration branch where weekly assignments, functional updates, and modules are consolidated before production deployment.
--sensor-module: A dedicated feature branch intended for developing, testing, and isolating source code related to peripheral sensors (e.g., DHT11, ultrasonic, LDR).
--wireless-module: A dedicated feature branch focused on network connectivity configurations, cloud telemetry, and wireless protocols (e.g., Wi-Fi, MQTT, Bluetooth).
